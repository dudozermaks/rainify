## How-to build and run (UNIX only):

### To build:

Deps:

* [CMake](https://cmake.org/)
* [ncurses](https://en.wikipedia.org/wiki/Ncurses) (most unix-based systems already have)

1. Clone repo
2. Go to repo dir
3. `mkdir build && cd build`
4. `cmake ..`
5. `cd .. && cmake --build build/`

### To run:

1. `./build/main`
